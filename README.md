# Mangoes 7

Timmy Tuff Knuckles' whole juiceline was pressed by the Mango clan. Your purpose? Hunt, juice, and blend mangoes.

## Description

This *experience* puts you in the boots of Timmy Tuff Knuckles, a peach who had his entire family blended in front of his eyes by the Mango clan. As Timmy Tuff Knuckles, seek your revenge in the heart of the Mango clan's base of operations. Point and click to puree the mangos, but be careful not to click on the yellow barrels, which contain a ripening agent which would decompose Timmy Tuff Knuckles nearly instantly.

## How to Run

1. Make sure you have **Python 3.13** installed.
2. Install the dependencies:
   run `pip install pygame-ce`
3. Run the game:
   run `python3 main.py`
